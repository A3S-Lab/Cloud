# AGENTS.md

## Configuration

- A3S ACL is the only product configuration language permitted in this
  repository.
- Never create, restore, document, or parse a non-ACL product configuration
  format or file extension.
- All product configuration files use the `.acl` extension.
- Parse and generate configuration only with `a3s-acl`; do not route ACL
  through a compatibility parser.
- Before completion, verify that source, tests, examples, and documentation
  contain only ACL configuration terminology and `.acl` paths.
